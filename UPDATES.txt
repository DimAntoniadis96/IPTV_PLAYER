Να θυμάσαι τι μάθαμε στην πορεία (θα σου χρησιμεύσει):

Install: πάντα με το sdb ... vd_appinstall (λόγω του κενού στο όνομα). Το Run του VS Code κάνει το Build, το install το κάνεις εσύ με την εντολή.
Μετά από κάθε αλλαγή: Build → η εντολή sdb → άνοιγμα από το τηλεκοντρόλ.
Η ταχύτητα κρατιέται με το flat θέμα (html.is-tv) — μη χαλάσουμε αυτό όταν πειράξουμε εμφάνιση.
